# 📝 Blogy

Blogy is a modern blog platform built with **Laravel** that allows users to create, manage, and explore articles .

---

## 🚀 Features

* 🔐 Authentication system (Login / Register)
* ✍️ Create, edit, and delete posts
* 🏷️ Categories and tags support
* 🔍 Search functionality
* 📄 Pagination for posts
* 🧑‍💻 Admin dashboard
* 📱 Responsive design

---

## 🛠️ Tech Stack

* **Backend:** Laravel (PHP)
* **Frontend:** Blade, HTML, CSS, JavaScript
* **Database:** MySQL

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/blogy.git
```

2. Navigate to project:

```bash
cd blogy
```

3. Install dependencies:

```bash
composer install
npm install
```

4. Setup environment:

```bash
cp .env.example .env
php artisan key:generate
```

5. Configure database in `.env`:

```
DB_DATABASE=blogy
DB_USERNAME=root
DB_PASSWORD=
```

6. Run migrations:

```bash
php artisan migrate
```

7. Start server:

```bash
php artisan serve
```

---

## 🔑 Usage

* Register a new account
* Login to dashboard
* Create and manage posts
* Browse posts by category or search
